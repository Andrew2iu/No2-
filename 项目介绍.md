# Vue2 文章管理系统

## 项目介绍

此系统覆盖文章管理前端应用与用户认证两大核心模块，实现从用户注册登录、文章发布编辑到分类管理的全流程功能，同时融入企业级开发规范，具备完整的用户权限控制和数据安全机制。

## 技术栈

-   **前端框架**: Vue 2.6.x
-   **UI 组件库**: Element UI
-   **状态管理**: Vuex
-   **路由管理**: Vue Router
-   **HTTP 请求**: Axios
-   **富文本编辑器**: Vue-Quill-Editor
-   **构建工具**: Vue CLI
-   **持久化存储**: vuex-persistedstate
-   **时间处理**: Day.js
-   **图表可视化**: ECharts

## 项目实现

### 1. 权限与安全机制

-   基于 Vue Router 实现路由权限控制，未登录用户无法访问受保护页面
-   配置路由白名单，确保登录/注册页可公开访问
-   通过 Axios 拦截器自动管理 token，实现 401 错误自动处理
-   使用 vuex-persistedstate 实现状态持久化，页面刷新后保持登录状态

### 2. 状态与路由管理

-   使用 Vuex 集中管理用户信息、token 等核心数据
-   配置路由懒加载优化初始加载速度
-   实现灵活的开发与生产环境构建配置

### 3. 核心业务功能

-   **文章管理**：支持分类管理、文章发布/编辑/删除，集成富文本编辑器与封面上传
-   **用户管理**：提供注册/登录、信息编辑、头像更新、密码修改等功能

### 4. 用户体验优化

-   集成 Element UI 表单验证，提供友好错误提示
-   响应式设计适配不同屏幕尺寸
-   统一的错误处理机制提升用户体验
